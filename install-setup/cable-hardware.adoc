= Conecte el hardware del sistema de almacenamiento ASA R2
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Conecta tu sistema de almacenamiento ASA r2 para habilitar la comunicación del clúster, el acceso a la red de host, la conectividad de gestión y las conexiones de bandejas de almacenamiento.



== Visión general del cableado de hardware

El cableado difiere según el modelo de plataforma porque cada modelo de ASA r2 tiene ubicaciones de ranura, asignaciones de puertos y configuraciones de módulos de E/S únicas. Selecciona el procedimiento para tu modelo de ASA r2:

* link:cable-hardware-a1k.html["Cable ASA A1K"]
* link:cable-hardware-a70-90.html["Cable ASA A70 y A90"]
* link:cable-hardware-a20-30-50.html["Cable ASA A20, A30 y A50"]
* link:cable-hardware-c30.html["Cable ASA C30"]


Vas a cablear cuatro tipos de conexiones durante el cableado de hardware:

* *Conexiones clúster/HA*
+
Conecta los dos controladores entre sí (clúster sin switches) o a switches de red de clúster (clúster conmutado). Estas conexiones permiten la comunicación del clúster y el failover de alta disponibilidad entre pares de controladores. El tráfico de interconexión de clúster y el tráfico de alta disponibilidad comparten los mismos puertos físicos.

* *Conexiones de red de host*
+
Conecta los controladores a hosts SAN usando Ethernet (iSCSI, NVMe/TCP) o Fibre Channel (FC, NVMe/FC) según la configuración de tu host. Estas conexiones proporcionan acceso a los datos para tus cargas de trabajo de almacenamiento.

* *Conexiones a la red de gestión*
+
Conecta los controladores a tu red de gestión para acceder a la administración del sistema a través de ONTAP System Manager y CLI.

* *Conexiones de estantería de almacenamiento*
+
Conecta los controladores a los estantes de almacenamiento NS224 para proporcionar capacidad de almacenamiento a tu sistema.





== Recursos adicionales

Los procedimientos de cableado muestran configuraciones comunes. El cableado específico depende de los componentes pedidos para tu sistema de almacenamiento. Para obtener detalles completos de configuración y prioridad de ranuras, consulta link:https://hwu.netapp.com["NetApp Hardware Universe"^].
